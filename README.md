##descripcion del proyecto
Este proyecto es una aplicación web interactiva que permite a los usuarios registrar datos como su nombre, correo, artista favorito y mensaje
a través de un formulario. Los datos se envían y almacenan en una base de datos MySQL mediante una API REST construida con Node.js y Express. 
Además, permite consultar y renderizar el último registro ingresado.
